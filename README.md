# Telegram Video Note Bot (tgbotcircle)

Aceasta este un bot de Telegram care transformă videoclipurile normale în mesaje rotunde (Video Notes).

## Cerințe
- Python 3.9+
- ffmpeg

## Instalare
```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

## Configurare
1. Creați un fișier `.env` pornind de la un template sau adăugând manual:
`BOT_TOKEN=token_aici`

## Rulare
```bash
python bot.py
```
